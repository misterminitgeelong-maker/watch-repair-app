# Product Update

## New in the Last Two Weeks

We shipped a major round of improvements focused on speed in the field, smarter job guidance, and day-to-day reliability.

### Mobile experience upgrades

- Faster, cleaner navigation on phones with a mobile-first layout pass across core screens.
- Better scheduling and dispatch usability with clearer job cards and in-context customer details.
- More practical in-field actions, including direct calling from relevant workflows.

### Smarter auto-key workflows

- AKL complexity now appears where technicians need it most.
- Known issues and recommended tools are surfaced directly in job context.
- Cutting machine profile guidance is available in flow to reduce guesswork.

### Better quoting consistency

- Quote suggestions now support pricing tiers (including B2B/tiered pricing) for more consistent outputs.

### Reliability and install improvements

- Deployment and startup stability improvements reduce interruptions.
- PWA install and app icon updates make the app easier to install and use on mobile devices.

## Why this matters

- Faster job completion in the field
- Fewer avoidable diagnostic/programming errors
- More consistent quotes and customer outcomes
- More confidence in daily app performance

Thanks for the continued feedback. More polish and workflow refinements are already in progress.
